3. 自适应学习率：针对参数的个性化步长

在复杂的神经网络中，有些参数更新频繁（梯度大），有些参数更新稀疏（梯度小）。对所有参数使用相同的学习率 是低效的。

B. RMSProp

为了解决 AdaGrad 的衰减问题，RMSProp 使用了指数移动平均来只保留近期梯度的平方信息：

$$E[g^2]t = \beta E[g^2]{t-1} + (1-\beta)g_t^2$$

这使得模型能够根据最近的梯度幅值动态调整步长。

5. 核心算法对比总结

优化器 核心特性 优点 缺点
SGD 简单一阶导 理论保证好，泛化潜力高 收敛慢，易陷于局部极小或振荡
Momentum 引入惯性 (EMA) 加速收敛，减小振荡 仍需人工精调学习率
RMSProp 自适应步长 解决梯度稀疏问题 学习率仍需初始设定
Adam 一阶 + 二阶矩 工业界默认，收敛极快，对超参数不敏感 在某些场景下泛化性可能略差于精调后的 SGD

为什么不用二阶优化（如牛顿法）

你可能有疑问：既然一阶导（梯度）收敛慢，为什么不用 Hessian 矩阵（二阶导）？

  1. 计算复杂度：Hessian 矩阵的大小是 （ 是参数量）。对于拥有 参数的模型，存储和求逆这个矩阵在工程上是不可能的。

  2. 非凸性：在非凸表面，牛顿法容易冲向极大值点或鞍点。

总结：优化器本质上是在收敛速度、计算开销和泛化能力之间做权衡。Adam 及其变体（如 AdamW）通过对梯度的一二阶统计，在保持低计算成本的同时，实现了极高的搜索效率。

为什么需要它？

在病态曲面（如狭长的峡谷）中，梯度在垂直于峡谷的方向上变化剧烈，而在谷底方向变化微小。 通过累积，使得震荡方向的梯度正负抵消，而固定方向的梯度得到加强，从而加速收敛。

4. 为什么 Adam 的公式里有分母

这是 Adam 实现自适应学习率的关键。

  • 如果梯度一直很大： 也会很大，分母变大，导致实际步长变小，起到一种“阻尼”作用，防止跳过极小值。

  • 如果梯度很小或很稀疏： 很小，分母变小，导致实际步长相对放大，使得模型在平坦区域或更新较少的参数上依然能保持学习。

这种机制在本质上是对参数空间进行了重标度（Preconditioning），试图将复杂的非凸椭圆曲面转化为更接近圆形的曲面，从而让简单的梯度下降更加高效。
